# 🗂️ Gerenciador de Tarefas Interativo

Aplicação web desenvolvida para organização de tarefas, permitindo cadastro, definição de prioridade, controle de prazos e acompanhamento do status das atividades.

---

## 🚀 Funcionalidades

- ✔ Adicionar novas tarefas  
- ✔ Definir prioridade (Baixa, Média, Alta)  
- ✔ Definir prazo  
- ✔ Marcar tarefas como concluídas  
- ✔ Editar tarefas  
- ✔ Excluir tarefas  
- ✔ Filtro por status (pendentes/concluídas)  
- ✔ Filtro por prioridade  
- ✔ Busca por nome da tarefa  
- ✔ Painel com resumo de tarefas  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web, incluindo manipulação do DOM, organização de dados e criação de interfaces interativas.

Além disso, o sistema foi pensado para simular uma aplicação real, com foco na usabilidade e na organização de informações.

---

## 💡 Aprendizados

Durante o desenvolvimento, foram trabalhados:

- Manipulação do DOM com JavaScript  
- Estruturação de layout com HTML e CSS  
- Criação de interações e lógica de aplicação  
- Organização de funcionalidades em um sistema real
